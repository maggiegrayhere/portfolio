Gazetteer

Comprehensive Country Information Tool
www.maggiegray.co.uk/gazetteer
Tech Stack

HTML, CSS, JavaScript, Bootstrap, JQuery, PHP

Details

The Gazeteer is a web application designed to provide detailed information about selected countries through an intuitive and user-friendly interface. To enhance the user experience, the application features modals for easy access to weather forecasts, news, currency information, and Wikipedia entries. Various APIs are integrated to offer a wide range of information to users. Clean, simple buttons facilitate easy reading and interaction, while Leaflet and Open Maps form the foundation for the interactive maps, ensuring a comprehensive and engaging user experience.

Company Directory

Interactive Personnel Management Tool
www.maggiegray.co.uk/company-directory
Tech Stack

HTML, CSS, JavaScript, Bootstrap, JQuery, PHP, MySQL

Details

The Company Directory is an interactive web application designed to display and manage personnel, departments, and locations within a company. The user-friendly GUI frontend connects seamlessly with a database, allowing users to search for personnel, departments, and locations effortlessly. Filtered search results streamline the user experience. Modals are utilized to provide straightforward tools for adding or editing records, while intuitive buttons enable the deletion of records without dependencies.

Clock Block Challenge

Interactive Morning Routine Tool for Children
www.maggiegray.co.uk/clock-block
Tech Stack

HTML, CSS, JavaScript

Details

The Clock Block Challenge is an engaging web application designed to help children get ready in the morning on time. The application features a series of timed challenges for children to complete, earning points to encourage good time management and the completion of routine tasks that can otherwise become monotonous. Children tick off each challenge upon completion, racing against the clock. A scoreboard tracks their progress. Bright colors, sounds, and a highly visible clock were incorporated to appeal to young users, making the morning routine fun and interactive.

GeoDotter

Interactive Local Navigation Map
www.maggiegray.co.uk/geodotter
Tech Stack

HTML, CSS, JavaScript, Bootstrap, JQuery, PHP

Details

GeoDotter is an interactive map designed to help users navigate and explore local areas while away from home. The tool offers a simple and comprehensive way to learn about new surroundings. Leveraging Leaflet and Open Source Maps, GeoDotter features clean, simple buttons and monotone icons for ease of navigation. The map includes search facilities, enabling users to find various points of interest in the local area. The design is fully responsive, ensuring the tool is easy to use on mobile devices while on-the-go.
